# Como rodar

1 - Navegue até o diretório raiz do projeto  
``` cd path/to/trabalho1POO2 ``` 

2 - Compile os arquivos  
```javac trabalho/*.java trabalho/enums/*.java```

3 - Execute o programa principal  
```java trabalho.Main```
